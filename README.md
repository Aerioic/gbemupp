gbemupp is an original gameboy emulator! This emulator is still a work in progress, but it's fully capable of playing Gameboy Tennis and other games. The arrow keys are the Dpad and "A", "J", "K", and "S" on the keyboard corrospound to "A", "Start", "Select" and "B" on the gameboy respectively.

This emulator can be built on Linux. Use the package manager and get SDL2. Then run make. Then run the exacutable and enjoy your favourite game!
